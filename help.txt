===TR===
YARDIM MENÜSÜ

Admin Yönetimi
- admin list              → Admin listesini gösterir.
- admin add [ID]          → Bu sohbeti veya belirtilen ID'yi admin yapar.
- admin remove [ID]       → Admin listesinden çıkarır.
- admin reset             → Tüm admin listesini temizler. (Onay gerekir)

Beyaz Liste Yönetimi
- whitelist list          → Beyaz listedeki sohbetleri gösterir.
- whitelist add [ID]      → Beyaz listeye ekler.
- whitelist remove [ID]   → Beyaz listeden çıkarır.
- whitelist reset         → Beyaz listeyi tamamen temizler. (Onay gerekir)
- whitelist control       → Yeni sohbet kontrolünü aç/kapat.

AI Yönetimi
- aichat                  → AI sohbetini aç/kapat.
- model                    → Model komutları için kullanım yardımı gösterir.
- model list               → Global modeli ve sohbet bazlı override'ları listeler.
- model global [isim]      → Ana (global) modeli değiştirir.
- model chat [isim]        → Bu sohbetin modelini değiştirir.
- model chat [ID] [isim]   → Belirtilen sohbetin modelini değiştirir.
- model chat reset [ID]    → Sohbetin override'ını kaldırır, global'e döner.
- model installed          → Yüklü Ollama modellerini listeler.
- personality             → Bu sohbetin aktif kişiliğini ve global kişiliği gösterir.
- personality chat [metin]→ Sadece bu sohbet için kişiliği günceller.
- personality global [metin]→ Global kişiliği günceller.
- think                   → Düşünme mesajı durumunu ve metnini gösterir.
- think on/off            → Düşünme mesajını aç/kapat.
- think [metin]           → Düşünme mesajı metnini günceller.
- aierror                 → AI hata mesajını gösterir veya değiştirir.
- replymode               → AI yanıtlarını quoted-reply modunda aç/kapat.
- media                   → Görsel ve dosya okuma durumunu gösterir.
- media image             → Görsel okumayı aç/kapat.
- media file              → Dosya okumayı aç/kapat.

Hafıza
- clear                   → Kullanım listesini gösterir, hiçbir şey silmez.
- clear chat              → Bu sohbetin hafızasını temizler.
- clear chat [ID]         → Belirtilen sohbetin hafızasını temizler.
- clear all               → Tüm sohbetlerin hafızasını temizler. (Onay gerekir)

Sıfırlama
- reset                   → Kullanım listesini gösterir, hiçbir şey sıfırlamaz.
- reset settings          → Bu sohbetin TÜM özel ayarlarını global'e çevirir. (Onay gerekir)
- reset settings [ad]     → Tek bir ayarı sıfırlar (onay istemez).
                             ad: model, personality, noprefix, groupchat, debugchat, fixedchat
- reset all settings      → Her şeyi proje varsayılanlarına döndürür. (Onay gerekir)

Sistem Ayarları
- prefix                  → Mevcut prefix'leri gösterir.
- prefix main <p>         → Ana prefix'i değiştirir.
- prefix debug <p>        → Debug prefix'ini değiştirir.
- prefix ignore <p>       → Yoksayma prefix'ini değiştirir.
- fixedchat               → Botu sadece bu sohbete kilitler veya serbest bırakır.
- noprefix                → Prefix olmadan her mesaja yanıt verme modunu aç/kapat.
- noprefixall             → Tüm sohbetler için prefix'siz modu aç/kapat. (Whitelist kapalıyken açmak veya
                             açıkken whitelist'i kapatmak tehlikeli olduğu için onay gerekir.)
- groupchat               → Ortak hafıza grup modunu aç/kapat.
- groupchat [ID]          → Belirtilen grup için ortak hafıza modunu aç/kapat.
- groupchat list          → Ortak hafıza modu açık olan grupları listeler.
- debugchat               → Bu sohbeti debug kanalı olarak ayarlar.

Rate Limit
- ratelimit               → Hız limiti ayarlarını gösterir.
- ratelimit on/off        → Hız limitini aç/kapat.
- ratelimit tokens <n>    → Maksimum token sayısını ayarlar.
- ratelimit refill <sn>   → Token yenileme süresini ayarlar.
- ratelimit warn <sn>     → Uyarı cooldown süresini ayarlar.
- ratelimit message <...> → Uyarı mesajını değiştirir.

Bilgi
- info                    → Genel durum özetini gösterir.
- info chat               → Bu sohbetin detaylarını gösterir.
- info ai                 → AI ve rate limit ayarlarını gösterir.
- info system             → Sistem, prefix ve whitelist bilgilerini gösterir.
- help                    → Bu yardım menüsünü gösterir.
- help github             → Projenin GitHub linkini gönderir.
- helplang tr/en          → Yardım dilini değiştirir.

===EN===
HELP MENU

Admin Management
- admin list              → Shows the admin list.
- admin add [ID]          → Adds this chat or the given ID to admins.
- admin remove [ID]       → Removes from the admin list.
- admin reset             → Clears the entire admin list. (Requires confirmation)

Whitelist Management
- whitelist list          → Shows the whitelist.
- whitelist add [ID]      → Adds to the whitelist.
- whitelist remove [ID]   → Removes from the whitelist.
- whitelist reset         → Clears the entire whitelist. (Requires confirmation)
- whitelist control       → Enables/disables new chat control.

AI Management
- aichat                  → Enables/disables AI chat.
- model                    → Shows usage help for model commands.
- model list               → Lists the global model and per-chat overrides.
- model global [name]      → Changes the main (global) model.
- model chat [name]        → Changes this chat's model.
- model chat [ID] [name]   → Changes a specific chat's model.
- model chat reset [ID]    → Removes a chat's override, falls back to global.
- model installed          → Lists installed Ollama models.
- personality             → Shows this chat's active personality and the global personality.
- personality chat [text] → Sets this chat's personality only.
- personality global [text]→ Sets the global personality.
- think                   → Shows think message status and text.
- think on/off            → Enables/disables the think message.
- think [text]            → Updates the think message text.
- aierror                 → Shows or updates the AI error message.
- replymode               → Enables/disables quoted-reply mode for AI responses.
- media                   → Shows image and file reading status.
- media image             → Enables/disables image reading.
- media file              → Enables/disables file reading.

Memory
- clear                   → Shows usage list, clears nothing.
- clear chat              → Clears this chat's memory.
- clear chat [ID]         → Clears a specific chat's memory.
- clear all               → Clears all chat memories. (Requires confirmation)

Reset
- reset                   → Shows usage list, resets nothing.
- reset settings          → Resets ALL of this chat's own overrides to global. (Requires confirmation)
- reset settings [name]   → Resets a single setting (no confirmation needed).
                             name: model, personality, noprefix, groupchat, debugchat, fixedchat
- reset all settings      → Factory-resets everything to project defaults. (Requires confirmation)

System Settings
- prefix                  → Shows current prefixes.
- prefix main <p>         → Changes the main prefix.
- prefix debug <p>        → Changes the debug prefix.
- prefix ignore <p>       → Changes the ignore prefix.
- fixedchat               → Locks the bot to this chat or releases it.
- noprefix                → Enables/disables no-prefix mode for this chat.
- noprefixall             → Enables/disables no-prefix mode for ALL chats. (Requires confirmation when
                             enabling while whitelist is off, or when disabling the whitelist while this is on.)
- groupchat               → Toggles shared-memory group chat mode.
- groupchat [ID]          → Toggles shared-memory mode for the given group ID.
- groupchat list          → Lists all groups with shared-memory mode enabled.
- debugchat               → Sets this chat as the debug channel.

Rate Limit
- ratelimit               → Shows rate limit settings.
- ratelimit on/off        → Enables/disables rate limiting.
- ratelimit tokens <n>    → Sets the max burst token count.
- ratelimit refill <sec>  → Sets the token refill interval.
- ratelimit warn <sec>    → Sets the warning cooldown.
- ratelimit message <...> → Updates the warning text.

Info
- info                    → Shows a quick status overview.
- info chat               → Shows this chat's details.
- info ai                 → Shows AI and rate limit settings.
- info system             → Shows system, prefix, and whitelist info.
- help                    → Shows this help menu.
- help github             → Sends the project's GitHub link.
- helplang tr/en          → Changes the help language.
